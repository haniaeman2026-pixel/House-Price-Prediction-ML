
# House Price Prediction Using Machine Learning

## Overview
This project predicts house prices using Machine Learning Regression Models.

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Conclusion
The dataset showed weak correlation with the target variable, resulting in low model performance and negative R2 scores.
